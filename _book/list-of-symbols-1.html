D420: Mathematics for Machine Learning

Chapter 1 List of Symbols

Symbol Meaning Example / Notes
\(x\) A scalar (real number) \(x \in \mathbb{R}\)
\(\mathbf{x}\) A vector \(\mathbf{x} = [x_1, x_2, \dots, x_n]^T\)
\(\mathbf{A}\) A matrix \(\mathbf{A} \in \mathbb{R}^{m \times n}\)
\(\mathbb{R}\) Set of real numbers \(\mathbb{R} = (-\infty, \infty)\)
\(f(\mathbf{x})\) A function mapping vectors to scalars Example: \(f(\mathbf{x}) = \mathbf{w}^T \mathbf{x} + b\)
